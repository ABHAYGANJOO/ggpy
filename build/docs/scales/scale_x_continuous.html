scale_x_continuous¶

scale_x_continuous scale a continuous x-axis. Its parameters are:

  • name - axis label
  • breaks - x tick breaks
  • labels - x tick labels
In [2]:
ggplot(diamonds, aes(x='carat', y='price')) + \
    geom_point() + \
    scale_x_continuous("Carat (1 = 200mg)")
